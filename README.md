# Vityarthi-project-world-clock
A simple world clock made by python

World Clock System

Description:-
Python-based app displaying current time and date in multiple cities with time zone support using Tkinter GUI and pytz. Auto-refreshes every second.
Author
- Name: Yatharth Shukla
- Reg No: 25BAI10751
- Branch: CSE-AIML

Objectives:-

- Build easy-to-use world clock in Python.
- Handle time zones with pytz.
- Create interactive GUI with Tkinter.
- Auto-refresh times.
- Manage errors like invalid zones.

 Requirements:-

- Python 3.8+
- pytz library
- Tkinter (built-in)
- IDE: VS Code, PyCharm, or IDLE
- OS: Windows, macOS, Linux

 Installation:-

1. Clone repo: git clone <repo-url>
2. Install pytz: pip install pytz
3. Run: python world_clock.py


Features:-

- Real-time time display.
- Multiple city support.
- Error handling for invalid zones.
- Auto-update every second.
